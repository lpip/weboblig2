Hva er GitHub?

GitHub er et web-basert oppbevaringsted som er oftest brukt for koding. Det tilbyr en distribuert versjonkontroll slik at programmere ikke behøver å bruke samme lokal nett for å jobbe på det samme prosjektet. Du får også en timestamp som viser når endringen er gjort eller hvem som har gjort endringen slik blir det lett å jobbe på et prosjekt sammen i grupper.

Scalable Vector Graphics (SVG)

Hva er SVG?

SVG er et XML-basert filformat for mark-up språk som skal beskrive 2D vektorer som også støtter interaktivitet og animasjon. SVG bilder er definert i XML filer som betyr at de kan bli søkt, skriptet og kompressert. SVG kan bli lett laget med hvilket som helst Text Editor program, også tegneprogram. Alle de største nettlesere støtter SVG.

Mathematical Markup Language (MathML)

Hva er MathML?

Hovedsaken bak MathML er å tilrettelegge bruken av matematiske og vitenskapelige innhold på nett. MathML kan bli brukt til å kode en presentasjon av matematisk betegnelse for visuell visning. MathML er designet for å dekke to formål, markup for strukturering av matematiske uttrykk og markup for presentasjon av matematiske uttrykk.
