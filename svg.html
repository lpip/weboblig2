Scalable Vector Graphics (SVG)

Hva er SVG?

SVG er et XML-basert filformat for mark-up språk som skal beskrive 2D vektorer som også støtter interaktivitet og animasjon. SVG bilder er definert i XML filer som betyr at de kan bli søkt, skriptet og kompressert. SVG kan bli lett laget med hvilket som helst Text Editor program, også tegneprogram. Alle de største nettlesere støtter SVG.

Eksempler på SVG

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
</svg>

Hvorfor SVG?

Fordi SVG-filer kun er tekst, tar de ikke så mye plass og kan redigeres med en vanlig teksteditor om ønskelig dette betyr at SVG-bildet kan skaleres uendelig uten at det gjør noe med kvaliteten.
